## 关于Linux环境配置和容易用到的指令
	

**关于如何使用Ubuntu来完成虚拟机的配置，可以看视频**==BV1DaxJzpEVm==，里面有相当详细的教程，从下载到基础的系统环境配置，以及Vscode和anaconda的安装
下面是一些基础的Linux指令及其功能

==ls​== 列出目录内容 ==ls -la：==以列表形式显示所有文件（包括隐藏文件）及详细信息
==cd​==切换工作目录==cd ~：==切换到用户家目录==cd ..：==切换到上级目录
==mkdir==​ 创建新目录

	
==mkdir new_folder==
==mkdir -p path/to/nested/folders：==创建多级目录


==rm==​删除文件或目录==rm file.txt==：删除文件

==cp==复制文件或目录

==mv==​移动或重命名文件/目录
==mv== old.txt new.txt：重命名
