## 关于Linux环境配置和容易用到的指令
	

**关于如何使用Ubuntu来完成虚拟机的配置，可以看视频**==BV1DaxJzpEVm==，里面有相当详细的教程，从下载到基础的系统环境配置，以及Vscode和anaconda的安装
下面是一些基础的Linux指令及其功能

`ls​`列出目录内容 

`ls -la:`以列表形式显示所有文件（包括隐藏文件）及详细信息

`cd​`切换工作目录`cd ~：`切换到用户家目录`cd ..：`切换到上级目录

`mkdir new_folder`创建新目录
`mkdir -p path/to/nested/folders：`创建多级目录

`rm`​删除文件或目录`rm file.txt：`删除文件

`cp`复制文件或目录

`mv=`移动或重命名文件/目录
`mv old.txt new.txt：`重命名

在权限管理上==sudo==拥有至高地位

使用`ping (要连接之物的ip地址）`来达成虚拟机和主机之间的连接（需要各自的ip地址）另外，在测试连接中，不要忘了按CTRL+C取消连接，否则愚蠢的虚拟机会一直试图连接下去。

###这边还附上了常用（存疑）的快捷键 *学！习！*

Ctrl + A​ 将光标快速移动到行首。

Ctrl + E​ 将光标快速移动到行尾。

Alt + B​ 将光标向后移动一个单词。

Alt + F​ 将光标向前移动一个单词。

Ctrl + U​ 剪切从光标处到行首的所有内容。

Ctrl + K​ 剪切从光标处到行尾的所有内容。

Ctrl + W​ 剪切光标前的一个单词。

Ctrl + Y 粘贴最近一次剪切的内容。

Ctrl + _​ 撤销上一次的编辑操作。

另外，Ctrl + R 可以让你搜索之前的指令，或者使用上↑下↓方向键可以让你以历史顺序找到之前所用过的指令

关于nano的简单使用
文件操作:​
`Ctrl + O`  保存文件。
`Ctrl + X`  退出 nano。
方向键实现上下左右移动光标。
`Ctrl + A/ Ctrl + E`光标跳到行首​ / 行尾。
`Ctrl + Y/ Ctrl + V`向上​ / 向下翻页。
`Ctrl + _`然后输入行号跳转到指定行。
`Ctrl + K`剪切当前行（或选中的文本块）。
`Ctrl + U`粘贴被剪切或复制的文本。
`Backspace/ Ctrl + D`删除光标前​ / 下的字符。
`Alt + U/ Alt + E`撤销​ / 重做上一次操作。
`Ctrl + W`查找文本，输入后按 Enter。
`Alt + W`查找下一个匹配项。
`Alt + >/ Alt + <(或 Ctrl + →/ Ctrl + ←)`在同时打开的多个文件标签页间切换。
帮助​`Ctrl + G`
